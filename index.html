Summarizing Code by Sequences of Non-Terminal Nodes of Abstract Syntax Tree for Improving Code Search

  • Introduction

Information of Abstract Syntax Tree (AST) has been researched and applied in many applications such as code search. One of the well-known strategy for taking advantages of AST in code search is to mining useful information about Data flow Graphs (DFG)s' nodes and edges. Prior works GraphCodeBERT and Unixcoder have been relied on this direction. In this work, we considered the other resources of AST that might be useful in code search: the non-terminal nodes. Our main idea is that while source code can be represented by a sequence of tokens as terminal nodes of AST, information about the non-terminal nodes of the same level can be considered as a type of summarization of that code snippet. We propose TreeMiner, a tool for augmenting features to the original embedding of GraphCodeBERT and Unixcoder to improve its accuracy by the summarized information of AST's non terminal nodes. By the evaluation of the recent curated CAT dataset, we show that TreeMiner can improve the accuracy of current state-of-the-art works GraphCodeBERT and Unixcoder by 2.5\% in various embedding sizes.
